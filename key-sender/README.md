# BYD Sub AI Key Sender

This is a Vite + React + TypeScript web application that generates and transmits GGwave sound signals to configure API keys on the **BYD Sub AI** Android application.

## Run Locally (로컬 실행)
1. Install dependencies (의존성 패키지 설치):
   ```bash
   npm install
   ```
2. Run the local development server (개발 서버 시작):
   ```bash
   npm run dev
   ```

## Production Build (프로덕션 빌드)
To build the application for deployment (배포용 빌드):
```bash
npm run build
```
